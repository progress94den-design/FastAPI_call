## FastAPI File Manager

Простое REST API приложение на FastAPI для работы с текстовыми файлами.

### Задание

Необходимо доработать существующий код, чтобы реализовать функционал Create, Read для текстовых файлов.

### Требования к доработке:

#### 1. Доделать весь функционал

- **GET /** – возвращает список имён всех файлов в папке /files (возвращать надо без формата файла)
- **GET /{file_name}** – чтение файла
- **POST /** – создание нового файла

#### 2. Добавить валидацию Pydantic

- Создать Pydantic модели для валидации входящих данных)

#### 3. Написать тесты

Используя `pytest` и `httpx`, написать тесты для всех эндпоинтов

## Инструкция по запуску

#### 1. Клонировать репозиторий
git clone git@github.com:progress94den-design/FastAPI_call.git

cd <название-папки>

#### 2. Установить зависимости через Poetry
poetry install

#### 3. Запустить приложение
poetry run python main.py
